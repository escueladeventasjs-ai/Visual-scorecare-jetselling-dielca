# DIELCA Live Scoreboard Final Secreto · Moderador 60%

Cambios:
- Moderador usa los mismos criterios que el jurado según fase:
  - Prospección: criterios de prospección.
  - Negociación: criterios de negociación.
- La pantalla de evaluación del moderador es privada:
  - Al entrar en modo moderador, se oculta ranking público.
  - Al volver, se muestra la pantalla pública.
- Ranking final:
  - Jurado 40%.
  - Moderador 60%.
- Premio del Público:
  - Solo jurado.
